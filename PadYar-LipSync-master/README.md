##🎯 هدف پروژه
پروژه PADYAR یک سیستم کامل برای تولید آواتارهای سخنگو با لب‌خوانی دقیق و هماهنگ ارائه می‌دهد. این سامانه با ترکیب مدل MuseTalk برای همگام‌سازی لب‌ها و موتورهای TTS (تبدیل متن به گفتار) مانند GPT، توانایی تولید ویدیوهای سخنگو از ورودی متنی را دارد.

##⚙️ ویژگی‌ها
تبدیل متن به گفتار با استفاده از API مبتنی بر GPT و همچنین مدل‌های محلی (آفلاین)

همگام‌سازی دقیق لب‌ها با صدای تولیدشده با کمک MuseTalk

رابط کاربری گرافیکی تحت وب با قابلیت تونل‌سازی از طریق Ngrok

پشتیبانی از اجرای کامل فرایندها در قالب نوت‌بوک Jupyter

قابلیت استقرار آسان در محیط‌های توسعه شخصی

##🧱 ساختار پروژه
## 🗂️ ساختار پوشه‌های پروژه

```bash
devenv-master/
├── MuseTalk_1.5/                 # مدل MuseTalk نسخه 1.5
├── MuseTalk_1/                   # نسخه پیشین MuseTalk
├── avatar_codes/                # کدهای آواتارها
├── final site/                  # رابط کاربری نهایی
├── site/                        # رابط کاربری تحت توسعه
├── v15/                         # شامل نوت‌بوک‌های اجرایی
│   └── Stream_Final_models15.ipynb  # نوت‌بوک اصلی برای اجرای مدل
├── Stream_Final_models.ipynb    # نسخه دیگر نوت‌بوک مدل‌ها
├── index.html                   # صفحه اصلی وب (احتمالاً برای پیش‌نمایش)
├── realtime_inference.py        # اسکریپت اجرای inference به‌صورت زنده
├── realtime.yaml                # فایل پیکربندی inference زمان واقعی
├── requirements.txt             # وابستگی‌های پایتون
├── README.md                    # فایل راهنما و معرفی پروژه
└── __init__.py                  # فایل اولیه پایتون (مناسب برای ساخت ماژول)
```

##🧪 راه‌اندازی محیط توسعه
1. نصب وابستگی‌ها
2. 

`pip install -r requirements.txt`
⚠️ توصیه می‌شود از محیط مجازی (virtualenv) یا Conda استفاده شود.

2. اجرای نوت‌بوک اصلی

فایل زیر شامل تمام مراحل آماده‌سازی و اجرای مدل‌هاست:

`v15/Stream_Final_models15.ipynb`
3. فعال‌سازی رابط کاربری (اختیاری)
برای راه‌اندازی UI تحت وب با استفاده از Ngrok:

`ngrok http 7860`

##🔍 نحوه عملکرد
کاربر متنی را به عنوان ورودی وارد می‌کند.

متن توسط یکی از روش‌های زیر به گفتار تبدیل می‌شود:

API GPT-Based (کیفیت بالا)

مدل آفلاین محلی

فایل صوتی تولیدشده توسط MuseTalk پردازش شده و ویدیو با لب‌خوانی دقیق ایجاد می‌شود.

(اختیاری) این ویدیو در رابط کاربری وب نمایش داده می‌شود.


##🧠 مدل‌های استفاده‌شده


| مدل          | کاربرد                       |
| ------------ | ---------------------------- |
| MuseTalk     | همگام‌سازی لب‌ها با صدا      |
| GPT TTS      | تبدیل متن به گفتار           |
| مدل‌های محلی | اجرای آفلاین و شخصی‌سازی‌شده |



##🛠️ تنظیمات زمان واقعی

فایل realtime.yaml شامل تنظیمات مدل‌ها برای اجرای سریع inference است. می‌توانید پارامترها را متناسب با سخت‌افزار و نیاز پروژه تنظیم کنید.
