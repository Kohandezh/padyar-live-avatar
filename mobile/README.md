# Padyar Mobile

This folder contains Padyar mobile SDKs.

- `mobile/padyar-android` — Android SDK
- `mobile/padyar-ios` — iOS SDK

## Boundary

The runtime package is `src/padyar_live/`.

Mobile code must not be imported by the runtime layer.
